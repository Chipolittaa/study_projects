Программа для обработки матриц, разработанная в рамках лабораторной работы №2. Матрица представляется как массив указателей на столбцы, и её элементы преобразуются на основе заданных характеристик. Характеристика столбца вычисляется как результат поразрядной операции "стрелка Пирса" между элементами столбца, после чего выполняется циклический сдвиг столбца на количество единиц в старшем байте характеристики. 

Для корректной работы программы необходим компилятор GCC версии 10.0 или выше и Linux с ядром версии 5.0 или выше. Программа поддерживает как случайное заполнение матрицы, так и ручной ввод элементов. Для запуска программы используйте команду:
./prg2psmN3146 [число_строк] [число_столбцов].

Для ручного ввода матрицы добавьте флаг -m:
./prg2psmN3146 -m [число_строк] [число_столбцов].

Для вывода дополнительной отладочной информации, включая характеристики столбцов и величины сдвига, перед запуском программы установите переменную окружения LAB2DEBUG:
export LAB2DEBUG=1
./prg2psmN3146 [число_строк] [число_столбцов]
unset LAB2DEBUG.

Для сборки программы выполните команду make all, а для удаления временных файлов make clean. Программа протестирована с различными матрицами и корректно обрабатывает граничные случаи.
