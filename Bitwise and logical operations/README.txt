
## Программирование на языке C для ОС Linux
### Описание программы
Программа изменяет порядок следования битов в числе на
обратный.
### Компиляция и запуск
Для сборки программы выполните команду:
```bash
make
```
Для запуска программы с числом в десятичной системе:
```bash
./prg1pmsN3146 <число>
```
Если аргумент не указан, будет сгенерировано случайное число.

### Очистка проекта
Чтобы удалить скомпилированные файлы:
```bash
make clean
```
